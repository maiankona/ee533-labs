# Lab 1 Code Instructions

## Installation

Install packages natively on MacOS according to the lab document. 

If you want to run it on a VM you must edit the Shell script from “./ns3 run” to “./waf --run"


## Deployment

1. Download Lab1Run.sh to ns-3.36.1 folder

2. cd scratch and download lab1.cc and evalMatrixLab1.py to the folder

3. Make it an executable using 

```bash
chmod a+x Lab1Run.sh
```
4. Run ./Lab1Run.sh

5. cd scratch and run

```
python3 evalMatrixLab1.py
```
6. Look in scratch folder for lab1results.txt and evaluation matrix pngs
